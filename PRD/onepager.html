ONE PAGE LyricsSync CONCEPT

1. 서비스명 & 슬로건

  • 서비스명: LyricsSync
  • 슬로건: Weird Translation Lyrics Quiz

2. 문제 정의

  • 설치형 파티 게임의 접근성 한계.
  • 친구들과 즉시 즐길 수 있는 가볍고 재미있는 웹 기반 파티 게임의 부재.

3. 핵심 아이디어

  • 설치 없이 언제 어디서든 친구들과 간편하게 즐기는 즉각적인 접근성. (Web/Mobile 접근성 + Socket.IO 기반 실시간 멀티플레이)
  • 음악적 감각과 추리력을 동시에 자극하는 '직역된 이상한 가사'라는 독특한 컨셉의 웹 퀴즈.

4. 주요 기능

  • 1. 링크 기반 즉시 참여 게임방
  • 2. 기계 번역된 가사 제공(연도, 특정 장르 선택 가능)
  • 3. 실시간 정답 판정 및 점수판괴 순위 표시
  • 4. 다양한 게임 모드(개인전, 팀전)
  • 5. 노래 제목 자동완성 기능

5. 기술 구조

  • Frontend: React + Vite
  • Backend: Node.js, Express, Socket.IO
  • Database: MongoDB
  • Hosting: Vercel/Render (Frontend), MSAZURE/AWS/GCP (Backend)

6. 기대효과

  • 끄투온라인, GarticPhone과 같은 틈새 시장 공략.
  • 친구들과의 즉각적인 접근성과 간편한 게임 참여로 높은 사용자 참여 유도.
  • 다양한 연령대와 음악 취향을 아우르는 폭넓은 사용자층 확보 가능.

7. UX

  • 1. 인식: 링크 공유
  • 2. 탐색: 방 입장/닉네임 입력
  • 3. 행동: 퀴즈 풀이/정답 제출
  • 4. 피드백: 실시간 점수/최종 순위 확인
  • 방장: 게임방 생성 → 방 링크 복사 → 친구들에게 공유 → 게임 시작.
  • 참가자: 링크 접속 → 닉네임 입력 → 게임 대기 → 게임 시작.
  • 공통: 퀴즈(번역 가사) 확인 → 채팅/입력창에 정답 입력 → 실시간 점수 확인 → 최종 순위 확인.

8. UI

  • 메인 화면: 게임방 생성 및 참여 버튼, 간단한 설명.
  • 게임 대기실: 참가자 목록, 게임 시작 버튼(방장 전용).
  • 퀴즈 화면: 번역된 가사 표시, 입력창, 실시간 점수판.
  • 최종 결과 화면: 최종 순위 및 점수 표시, 다시 하기 버튼.
  • 반응형 디자인: 모바일 및 데스크탑 모두 최적화.
